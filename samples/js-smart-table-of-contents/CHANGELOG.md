# Changelog

## v2.3.0

### Added

- ScrollSpy active section tracking.
- Floating drawer navigation mode.
- Inline table of contents mode.
- Live heading search filter.
- Estimated reading time and section count metadata.
- Dynamic content refresh detection after SharePoint page hydration.
- Nested heading support for Heading 1 through Heading 4.
- Configurable maximum heading depth.
- Configurable scroll offset.
- Optional section numbering.
- Optional empty-state message.

### Improved

- Smooth scrolling behavior.
- Active heading synchronization.
- Heading ID generation and targeting stability.
- Nested heading expansion on hover, focus, and active section.
- Drawer layout containment.
- Header and search box layout.
- Modern SharePoint visual alignment.
- Non-invasive navigation without URL hash rewriting.

### Fixed

- Hover overlap between nested headings and search/header region.
- Drawer scroll region containment.
- Header z-index layering issues.
- Delayed heading detection on modern SharePoint pages.
- Duplicate or unstable heading target behavior.

## v1.0.0

### Added

- Initial Smart Table of Contents SPFx web part.
- Basic heading detection.
- Basic SharePoint page navigation.
